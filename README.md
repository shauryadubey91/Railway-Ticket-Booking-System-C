#Railway Ticket Booking System (C)

A console-based Railway Ticket Booking System developed in C language.
This project simulates basic railway reservation functionalities.

#Features
- User Login / Registration
- Book Railway Tickets
- View Booked Tickets
- Cancel Tickets
- File handling for data storage

#Technologies Used
- C Programming Language
- File Handling, Structures, Functions

#How to Run
```bash
gcc src/*.c -o railway
./railway
